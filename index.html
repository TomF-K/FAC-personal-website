I'm Tom, a frontend developer creating responsive, accessible sites using HTML, CSS and JavaScript.

For the past 15 years I've been working as a professional musician and music teacher, performing across the UK and Europe and working with students of all ages and abilities.

Since becoming a parent my priorities have shifted; I want a career which allows me to spend more time at home and has a more consistent schedule than being a performing musician. That's where web development comes in.

Over the past two years I've been working through courses including Harvard's CS50, Free Code Camp, Frontend Masters, and The Odin Project to improve my frontend development skills. This site showcases some of what I've learned about responsive design, you can view the Github repo here.
